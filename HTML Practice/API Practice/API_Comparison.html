Setup Requirements

  1. Include Cesium library CDN in your HTML
  2. Set up a Cesium Ion token (for Test1.html) or use OpenStreetMap (for Test1_NoToken.html)
  3. Use the updated terrain property instead of terrainProvider
  4. Serve files through HTTP server for best results
